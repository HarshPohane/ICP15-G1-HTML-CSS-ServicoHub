# Servico Hub

Servico Hub is a home service website project built using **HTML** and **CSS**.  
It helps users find and book trusted local service professionals like **Electricians, Plumbers, AC Repair Technicians, and Carpenters**.

## Features

- Clean and responsive website design
- Simple navigation bar
- About Us page
- Service categories
- Contact section
- Booking page
- Footer with quick links and contact details

## Pages

- Home Page
- About Us Page
- Services Page
- Contact Page
- Booking Page

## Technologies Used

- HTML5
- CSS3

## Services Provided

- Electrician
- Plumber
- AC Repair
- Carpenter
